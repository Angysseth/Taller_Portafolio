    3. Realizar un programa en el cual se solicite el nombre, apellidos y edad de la persona. Si la edad de la persona es menor que 18 años, entonces, deberá imprimirse el siguiente mensaje: [Nombre completo] usted es menor de edad, por lo tanto, no puede entrar a la fiesta, por favor devuélvase a su casa.

                     *
                     * @author user
                     */

                            System.out.println("Digita tus apellidos");
                            String apellidos = input.nextLine();

El profesor de geometría está explicando a sus estudiantes las fórmulas para calcular el área de diferentes figuras geométricas, para ello requiere una aplicación que le facilite el ejercicio solicitándole los valores al estudiante. La aplicación debe permitir que el estudiante seleccione si desea calcular el área de un rectángulo, triángulo o trapecio. No olvide solicitar los datos necesarios para realizar cada cálculo y mostrar su respectivo resultado. 

                        

                    package com.mycompany.taller2;

                    import java.util.Scanner;
                    
                    /**
                     *
                     * @author user
                     */
                    public class Taller2_P9 {
                    
                        public static void main(String[] args) {
                            Scanner input = new Scanner(System.in);
                            System.out.println("Calcula el área para las figuras geométricas");
                            System.out.println("Selecciona la figura de la cual deseas calcular el área");
                            System.out.println("1. Rectángulo");
                            System.out.println("2. Triángulo");
                            System.out.println("3. Trapecio");
                            int opcion = input.nextInt();
                    
                            switch (opcion) {
                                case 1:
                                    System.out.println("Digita la base del rectángulo en centimetros");
                                    int base = input.nextInt();
                                    System.out.println("Digita la altura del restángulo en centimetros");
                                    int altura = input.nextInt();
                                    double area = base * altura;
                                    System.out.println("El área del rectángulo es: " + area + " cm^2");
                                    break;
                                case 2:
                                    System.out.println("Digita la base del triángulo en centimetros");
                                    base = input.nextInt();
                                    System.out.println("Digita la altura del triángulo en centimetros");
                                    altura = input.nextInt();
                                    area = (base * altura) / 2;
                                    System.out.println("El área del triángulo es: " + area + " cm^2");
                                    break;
                                case 3:
                                    System.out.println("Digita la base 1 del trapecio en centimetros");
                                    base = input.nextInt();
                                    System.out.println("Digita la base 2 del trapecio en centimetros");
                                    int baseDos = input.nextInt();
                                    System.out.println("Digita la altura del trapecio en centimetros");
                                    altura = input.nextInt();
                                    area = ((base + baseDos) * altura) / 2;
                                    System.out.println("El área del trapecio es: " + area + " cm^2");
                                    break;
                                default:
                                    System.out.println("La opción no es válida");
                            }
                    
                        }
                    
                    }

El banco "Su banco fiel" es un banco que inicia sus actividades financieras y necesita una aplicación para llevar las cuentas de sus usuarios; por lo tanto, se sugiere que la cuenta tenga los atributos titular y cantidad. Para cada cliente las cuentas permitirán realizar ingresos, retiros o consultas de valor. En los ingresos no se pueden insertar valores negativos y para los retiros el valor no puede ser mayor al valor que tiene en la cuenta. 

                        
	
                    package com.mycompany.taller2;

                    import java.util.Scanner;
                    
                    /**
                     *
                     * @author user
                     */
                    public class Taller2_P10 {
                    
                        public static void main(String[] args) {
                            Scanner input = new Scanner(System.in);
                            int cuenta = 2000000;
                    
                            System.out.println("Hola, bienvenido(a) a Tu Banco Fiel");
                            System.out.println("Escoge una de la siguientes opciones");
                            System.out.println("1. Ingresos");
                            System.out.println("2. Retiros");
                            System.out.println("3. Consultar valor");
                            System.out.println("4. Salir del sistema");
                            int opcion = input.nextInt();
                            input.nextLine();
                    
                            switch (opcion) {
                                case 1:
                                    System.out.println("Digita el valor a ingresar");
                                    int ingreso = input.nextInt();
                                    cuenta += ingreso;
                                    if (ingreso < 0) {
                                        System.out.println("No es posible ingresar valores negativos");
                                    } else {
                                        System.out.println("Su ingreso de $" + ingreso + " Se realizó satisfactoriamente, Actualmente tienes $" + cuenta + " en tu cuenta.");
                                    }
                                    break;
                                case 2:
                                    System.out.println("Digita en valor que deseas retirar");
                                    int retiro = input.nextInt();
                                    cuenta -= retiro;
                                    if (retiro > cuenta) {
                                        System.out.println("Tu retiro no se puede realizar, ya que el valor es mayor al que tienes disponible en tu cuenta");
                                    } else {
                                        System.out.println("Tu retiro de $" + retiro + " fué exitoso. Actualmente tienes $" + cuenta + " en tu cuenta.");
                                    }
                                    break;
                                case 3:
                                    System.out.println("El valor en tu cuenta es de: $" + cuenta);
                                    break;
                                case 4:
                                    System.out.println("Has salido del sistema!");
                                    break;
                                default:
                                    System.out.println("La opción no es válida");
                            }
                        }
                    
                    }
